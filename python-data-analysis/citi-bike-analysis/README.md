# Citi Bike Usage Analysis (Power BI)

## Objective
To analyze Citi Bike usage patterns and identify insights related to
subscriber behavior, peak hours, and seasonal trends.

## Dataset
Public Citi Bike dataset containing ride duration, user type, age group,
start stations, and timestamps.

## Tools Used
- Power BI
- Power Query
- Excel

## Key Insights
- 98% of users are subscribers
- Peak usage during office commute hours
- Highest usage among 25–34 age group
- Usage drops significantly during extreme weather

## Outcome
Created an interactive dashboard and presented insights as part of
**Data Analytics using Power BI – LIVEWIRE certification**.
